# 👨‍💻 깊은 고민으로 세상을 바꾸고 싶은 이성준입니다
시스템이 실제로 어떻게 움직이는지 깊이 파악하고, 프로젝트를 통하여 세 차례의 개발 사이클을 직접 이끌며 풀스택 개발자로서 탄탄한 기반을 다져왔습니다.
운영체제, 자료구조 등 기본적인 CS 지식을 바탕으로 메모리와 데이터의 흐름, 시스템 내부 구조를 파악하며 개발합니다.
특히 800일 동안 1800개 이상의 알고리즘 문제를 해결하며(백준 다이아몬드 V) 최적화 역량을 길렀고, 실제 게임 프로젝트에서 시간복잡도를 O(N^2)에서 O(N)으로, 로딩 시간을 20초에서 3.6초로 개선한 경험이 있습니다.
업무의 알파부터 오메가까지 이해하고 주어진 분야의 최적화에 기여하는 개발자가 되고 싶습니다.<br>

### [ParaDICE] 로그라이크 RPG 게임 클라이언트
- **기간:** 2026.01 - 2026.04
- **역할 및 기술:** 게임 클라이언트 개발 (기여도 40%) / Unity
- **주요 성과:**

- **기간:** 2025.10 - 2025.11
- **역할 및 기술:** 인프라 배포 (기여도 40%), 풀스택 (기여도 10%) / Github Actions, NGINX, Docker, Deck.gl
- **주요 성과:**
  - Docker Compose, Nginx 리버스 프록시, Github Actions를 활용한 서비스 일괄 자동 배포 파이프라인 구축
  - Playwright 크롤링 및 Pillow Webp 변환으로 백그라운드 이미지 용량 75% 감소 최적화 (432KB → 110KB)
  - Deck.gl을 이용한 히트맵 생성 및 쿼리 최적화(캐싱, GROUP BY)를 통한 유저 방문 그래프 구현
